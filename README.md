# 🔎 프로젝트 설명
*같이 운동하면서 환경도 지켜볼래?
  
길을 걷다보면 길가에 무수히 많은 쓰레기들을 마주칠 수 있습니다. 초록초록은 길가의 쓰레기 근절과 더불어 운동효과까지
초록초록은 길을 걸으면서 쓰레기를 줍고 주변 쓰레기통의 위치를 확인하여 처리할 수 있는 앱을 제작하여 운동 효과와 더불어 일상속의 환경운동 효과도 기대할 수 있음
